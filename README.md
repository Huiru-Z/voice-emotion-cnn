# voice-emotion-recognize
这是一个课程作业，一个基于卷积神经网络的语音情感识别模型

## 技术路线
.wav音频文件 -> MFCC特征提取 -> CNN -> 情绪分类(愤怒、恐惧、开心、惊喜、悲伤、平静)

## 所用数据集
CASIA  
包含了两男两女的1200条语音样本，涵盖了愤怒、恐惧、开心、惊喜、悲伤、平静六种情绪，均为汉语  
简介博客 https://blog.csdn.net/gitblog_09722/article/details/143074127  
仓库 https://gitcode.com/Premium-Resources/fa78c

## 训练过程
### 1、数据集加载
程序首先遍历 CASIA 数据集目录  
  
读取所有 .wav 文件路径

根据文件夹名称映射为情绪标签
### 2、训练集与测试集划分
随机将数据的80%划分成训练集，20%作为测试集

### 3、语音特征提取
通过mfcc将wav文件转化成(1, 40, 300)的特征矩阵

### 4、构建神经网络

初始化卷积神经网络，包括：  

3层二维卷积层  

ReLU非线性激活函数  

最大池化层  

2层全连接层  

模型参数 𝜃随机初始化。  
### 5、损失函数
使用多分类交叉熵损失衡量预测情绪分布与真实标签之间的差异。

### 6、模型保存
训练结果保存在“emotion_cnn.pth”中

## 结果
训练12个epoch后损失率下降到0.04，自动终止训练  
使用自己录的声音测试结果正确
